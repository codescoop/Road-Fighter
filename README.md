# Road-Fighter
Road Fighter is a car-based arcade game developed using python.

## Usage

1. Open your Terminal/cmd

2. git clone this project to your preferred directory.

3. `cd Road-Fighter`

4. `python RoadFighter.py`

## Dependencies

- Pygame (http://pygame.org)
